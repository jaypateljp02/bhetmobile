# Bhet Mobile - Ratnagiri's Trusted Mobile Store

A modern, responsive React application for **Bhet Mobile**, a leading mobile phone retailer in Ratnagiri. This project replaces the previous static HTML site with a modular component-based architecture using Vite and Tailwind CSS.

- **Modern UI/UX**: Clean, professional design using a custom color palette (Brand Orange).
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Component-Based**: Modular structure with separate components for Navbar, Hero, Products, Services, etc.
- **Interactive Elements**:
  - Floating WhatsApp chat button.
  - Hover effects on product cards.
  - Mobile usage data stats.
- **Map Integration**: Embedded Google Maps showing the store's location at Navkar Enclave.
- **Tailwind CSS**: Utility-first styling for rapid UI development.
